Este proyecto es un CRUD hecho con Node y Express.
se hizo para practicar cómo manejar rutas y las operaciones básicas de agregar, leer, editar y borrar. 
En vez de usar una base de datos, todo se guarda en archivos JSON con fs. 
Se puede probar desde Postman mandando las distintas peticiones a las rutas de productos y carritos.
Es algo simple, más para entender la lógica y cómo levantar un servidor desde cero.
